Modrinth Guide

Install Modrinth here: https://modrinth.com/app installation is straight foward

Once opened and set up. If you are not already, log in to your minecraft account

Now import the modpack file (.mrpack) that you've downloaded, by creating an instance as shown

Now open the modpack and click on the gear icon, then allocate ram accordingly. Recommended to do at MOST 8GB if available and not more than half of your total ram.

Now you should be able to press play and run the modpack
